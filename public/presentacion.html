Contexto y necesidad

El negocio de GNL de Naturgy gestiona relaciones comerciales con contrapartes en más de 40 países. La complejidad del pipeline, los requisitos legales (KYC, NDA, MSPA) y la necesidad de información actualizada exigen una herramienta centralizada.

🌍

Dispersión geográfica

Contrapartes en Asia-Pacífico, Europa, América, Oriente Medio y África

📋

Documentación compleja

KYC, NDA y MSPA pendientes con múltiples entidades simultáneamente

📈

Pipeline activo

Decenas de oportunidades en distintas fases de negociación en paralelo

Stack tecnológico

Backend

⚙️Node.js + Express 5 — Servidor API REST
🗄️SQLite (better-sqlite3) — Base de datos embebida, WAL mode
🔐express-session — Autenticación con sesiones
📊xlsx — Importación de datos desde Excel
🤖OpenAI SDK — GPT-4o-mini para el asistente IA
🔄PM2 — Proceso persistente en servidor Ubuntu

Frontend

🌐HTML/CSS/JS vanilla — SPA sin frameworks, cero dependencias frontend
🗺️Leaflet + OpenStreetMap — Mapa mundial interactivo
🎨Font Awesome — Iconografía
📡Fetch API — Comunicación con el backend

Funcionalidades diferenciales

🔗

Iconos de enlace inteligentes

LinkedIn de contactos, documentos KYC/NDA/MSPA y fichas de riesgo país (CESCE) se muestran directamente como iconos clicables en las tablas

🌍

Selector de país con FK garantizada

El formulario de entidades usa un desplegable dinámico de países, evitando errores de integridad referencial

⚡

Código ISO auto-calculado

Al dar de alta un país, el código ISO 3166-1 alpha-3 se calcula automáticamente al escribir el nombre en español

📥

Importación Excel masiva

Carga de datos desde .xlsx con 5 hojas predefinidas respetando el orden de importación por FK
